# 📦 LogiExpress Colombia S.A.S.
### 🚚 Sistema de Gestión Logística - Mockup Interactivo

---

## 📋 Descripción
Prototipo interactivo del sistema transaccional diseñado para la empresa **LogiExpress Colombia S.A.S.**, una compañía dedicada a la prestación de servicios logísticos a nivel nacional. Este mockup simula las principales funcionalidades del sistema de gestión de envíos, clientes, vehículos, bodegas y facturación.

---

## 🚀 Cómo usar
1. 📥 Descargar el archivo `LogiExpress_Mockup.html`
2. 🌐 Abrirlo en cualquier navegador (Chrome, Edge, Firefox)
3. 👤 **Usuario:** `admin@logiexpress.com`
4. 🔑 **Contraseña:** `12345678`
5. ✅ Clic en **Ingresar** y navegar por los módulos

---

## 🧩 Módulos del Sistema

| Módulo | Descripción |
|--------|------------|
| 📊 **Dashboard** | Panel general con estadísticas de envíos activos, en tránsito, entregados y pendientes |
| 📦 **Envíos** | Gestión completa de envíos con número de guía, estado y seguimiento |
| 👥 **Clientes** | Registro y consulta de clientes con datos de contacto |
| 🚚 **Vehículos** | Control de flota vehicular con estado de disponibilidad |
| 🏭 **Bodegas** | Administración de bodegas con indicadores de ocupación |
| ➕ **Nuevo Envío** | Formulario de registro de nuevos envíos |
| 🧾 **Facturas** | Módulo de facturación con subtotal, IVA y total |
